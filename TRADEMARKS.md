# Local Coding Agent Trademark Policy

This policy separates the open-source license for the code from permission to
use the project's identity. It applies to the `Local Coding Agent` name, project
logo, banner, product icons, and other brand assets that identify the official
project maintained by Long Nguyen (`@LongNgn204`).

This policy does not state that a trademark registration has already been
granted. Do not use the registered trademark symbol (`R` in a circle) unless a
valid registration covering that use has been confirmed.

The owner-facing filing preparation steps are documented in
[docs/TRADEMARK-REGISTRATION-CHECKLIST.md](docs/TRADEMARK-REGISTRATION-CHECKLIST.md).

## Permitted use

You may:

- refer truthfully to the project in articles, reviews, tutorials, and
  compatibility statements;
- link to the official repository or releases;
- state that another project is based on, derived from, or compatible with
  Local Coding Agent, provided the statement is accurate and not misleading;
- retain copyright, license, and attribution notices required by the AGPL.

## Permission required

Written permission from the project owner is required to:

- use `Local Coding Agent` as the primary name of a fork, hosted service,
  commercial product, company, domain, social-media account, or app-store entry;
- use the official logo, banner, or product icon as the primary identity of a
  modified build;
- describe a third-party build or service as `official`, `certified`,
  `approved`, or `endorsed` by the project;
- use branding that is likely to make users confuse a third-party release with
  a release published by `LongNgn204`.

## Forks and modified distributions

Forking and modifying the Community Edition code is governed by the AGPL. A
public fork or redistributed modified build should use a clearly different
primary name and visual identity, preserve required notices, identify the
changes, and include a statement such as:

> Based on Local Coding Agent Community Edition. This is an independent,
> unofficial distribution and is not endorsed by the Local Coding Agent
> project.

The code license does not authorize impersonation or misleading use of project
branding. See [BRAND-GUIDELINES.md](BRAND-GUIDELINES.md) for practical examples.

## Reporting confusing use

Please report suspected impersonation or confusing branding privately to the
project owner through the contact method listed on the official GitHub profile:
<https://github.com/LongNgn204>.

Các bước chuẩn bị hồ sơ dành cho chủ dự án nằm tại
[docs/TRADEMARK-REGISTRATION-CHECKLIST.md](docs/TRADEMARK-REGISTRATION-CHECKLIST.md).

### Được phép

- Nhắc đến dự án một cách trung thực trong bài viết, đánh giá và hướng dẫn.
- Liên kết đến repository hoặc trang release chính thức.
- Nói một dự án được phát triển dựa trên hoặc tương thích với Local Coding
  Agent, miễn nội dung chính xác và không gây hiểu nhầm.

### Cần có sự cho phép bằng văn bản

- Dùng `Local Coding Agent` làm tên chính của fork, dịch vụ, sản phẩm thương
