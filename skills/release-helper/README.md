# release-helper

**English.** Cut a release safely: run
the agent/pro/security tests, validate skills, check version constants, and
update the changelog. Never publishes without maintainer approval. See
`SKILL.md` and `skill.json`.

**Tieng Viet.** Phat hanh ban release mot cach an toan: chay test
agent/pro/security, kiem tra skill, kiem tra hang so phien ban va cap nhat
changelog. Khong publish khi chua co phe duyet. Xem `SKILL.md` va `skill.json`.
