# repo-support

**English.** Orient a coding agent inside a workspace with the fewest tool
round-trips: snapshot first, search narrowly, patch with preview/validate, and
keep long output in local reports. See `SKILL.md` and `skill.json`.

**Tieng Viet.** Giup agent lam quen workspace voi it luot goi tool nhat: chup
snapshot truoc, tim kiem gon, sua bang preview/validate, va giu log dai trong
report cuc bo. Xem `SKILL.md` va `skill.json`.
