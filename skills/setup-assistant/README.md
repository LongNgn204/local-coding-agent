# setup-assistant

**English.** A copy-ready playbook for an AI coding agent that installs and
verifies Local Coding Agent for a customer. It uses the universal CLI, keeps
`mode=safe`/`policy=balanced`, and never touches secrets or the proprietary
tunnel client. See `SKILL.md` for the exact steps and `skill.json` for the
machine-readable manifest (commands + verify URLs).

**Tieng Viet.** Kich ban cho AI agent cai dat va kiem tra Local Coding Agent
cho khach hang. Dung CLI chung, giu `mode=safe`/`policy=balanced`, khong dong
vao secret hay tunnel client. Xem `SKILL.md` de biet cac buoc va `skill.json`
cho manifest (lenh + URL kiem tra).
