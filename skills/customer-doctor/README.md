# customer-doctor

**English.** Diagnose a broken customer setup without leaking secrets. Runs
status, network doctor, and the redacted support report, then reports the most
likely cause and the exact fix. See `SKILL.md` and `skill.json`.

**Tieng Viet.** Chan doan cai dat loi ma khong lo secret. Chay status, network
doctor va support report (da an secret), sau do bao nguyen nhan va cach sua.
Xem `SKILL.md` va `skill.json`.
