# Local Coding Agent Brand Guidelines

## Official identity

- Project name: **Local Coding Agent**
- Edition: **Community Edition**
- Stable public release: **v4.4.2-prodev**
- Official repository: <https://github.com/LongNgn204/local-coding-agent>
- Official publisher/maintainer: **Long Nguyen (`@LongNgn204`)**

Only artifacts published through the official repository and its release page
should be presented as official Local Coding Agent releases.

## Naming a fork

Choose a new primary name for a fork or modified distribution. A truthful
secondary statement is welcome, for example:

> Project Nova - based on Local Coding Agent Community Edition

Avoid names that differ only by punctuation, capitalization, `Pro`, `Plus`,
`Official`, or a version suffix. Those patterns can make users believe the fork
is an official edition or release channel.

## Logos and visual assets

Do not use the official logo, banner, icon, or a confusingly similar asset as
the primary identity of a fork or third-party product without written
permission. Screenshots used for commentary, tutorials, or compatibility
documentation should identify the official project accurately and should not
imply sponsorship.

## Attribution template

Use this notice for a modified distribution:

```text
This product includes software derived from Local Coding Agent Community
Edition, Copyright (c) 2026 Long Nguyen, licensed under AGPL-3.0-or-later.
This distribution is independent and is not endorsed by the Local Coding Agent
project. Local Coding Agent source: https://github.com/LongNgn204/local-coding-agent
```

## Hướng Dẫn Thương Hiệu - Tiếng Việt

- Tên dự án: **Local Coding Agent**
- Phiên bản sản phẩm: **Community Edition**
- Bản stable công khai: **v4.4.2-prodev**
- Repository chính thức: <https://github.com/LongNgn204/local-coding-agent>
- Chủ dự án: **Long Nguyễn (`@LongNgn204`)**

Chỉ artifact được công bố qua repository và trang release chính thức mới nên
được giới thiệu là bản phát hành chính thức của Local Coding Agent.

Fork hoặc bản phân phối đã sửa đổi cần có tên chính và nhận diện riêng. Có thể
dùng mô tả phụ trung thực như `được phát triển dựa trên Local Coding Agent
Community Edition`, nhưng không được khiến người dùng hiểu nhầm fork là bản
chính thức. Không dùng logo, banner hoặc icon chính thức làm nhận diện chính khi
chưa có sự cho phép bằng văn bản.

Mẫu ghi công cho bản phân phối đã sửa đổi:

```text
Sản phẩm này có sử dụng phần mềm phát triển từ Local Coding Agent Community
Edition, Copyright (c) 2026 Long Nguyễn, theo giấy phép AGPL-3.0-or-later.
Đây là bản phân phối độc lập, không được dự án Local Coding Agent bảo trợ.
Mã nguồn gốc: https://github.com/LongNgn204/local-coding-agent
```
